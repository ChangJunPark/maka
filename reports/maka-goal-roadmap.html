1. 우리의 최종 목표

목표 문장

Maka는 “로컬 문서 작업용 Markdown IDE”다. Typora처럼 문서 작성에 집중할 수 있고, VS Code처럼 실제 폴더와 터미널을 다루며, 외부 편집과 자동저장 사이에서 사용자의 데이터를 잃지 않는다.

Tauri React/TypeScript Local folder Markdown + Mermaid Integrated terminal v1 polish 진행 중

v1에서 일부러 뺀 것

  • 완전한 Typora식 inline WYSIWYG
  • 테마 커스터마이징
  • 고급 표 편집
  • 다중 탭
  • Outline/sidebar
  • PDF/export polish

6. 주요 리스크

실제 GUI 재시작 smoke

마지막 workspace 복원은 build 검증은 됐지만, 커밋 이후 실제 앱 재시작 수동 확인은 다음 라운드에서 해야 한다.

Mermaid bundle 크기

현재 build는 통과하지만 Mermaid 관련 chunk warning이 있다. lazy import로 초기 로딩을 줄이는 것이 좋다.

Watcher edge case

atomic save, rename, delete, 대형 workspace 이벤트는 더 많은 실제 폴더 smoke와 debounce가 필요하다.

7. 추천 다음 액션
